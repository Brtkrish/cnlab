=====================================
GO BACK N ARQ ALGORITHM
=====================================

SERVER (SENDER) ALGORITHM:
---------------------------
1.  Start
2.  Declare necessary variables including server_sock, acpt_sock,
    i, write_buff, read_buff, set and timeout
3.  Create a socket for TCP communication using socket(AF_INET, SOCK_STREAM, 0)
    If failed → print error and exit
4.  Set up the server address structure with appropriate values
    for family, port and IP address
5.  Bind the socket to the server address using bind()
    If failed → print error and exit
6.  Listen for incoming connections using listen()
    If failed → print error and exit
7.  Accept a connection from client using accept()
    If failed → print error and exit
8.  Initialize frame number i = 0
9.  While i < total number of frames:
    9.1  Prepare frame message "Frame: i"
    9.2  Send frame i to the client using send()
         → Print "Sent: Frame i"
    9.3  Send next frame (i+1) using send()
         → Print "Sent: Frame i+1"
    9.4  Start timer using select() with timeout = 2 seconds
    9.5  If timeout occurs:
         → Print "Timeout! Retransmitting from frame i"
         → Go back to frame i
         → Resend ALL frames from frame i onwards
    9.6  If ACK received from client using recv():
         → Print received ACK
         → Increment i
10. Close accepted socket and server socket using close()
11. Stop


CLIENT (RECEIVER) ALGORITHM:
-----------------------------
1.  Start
2.  Declare necessary variables including cli_sock, i, flag,
    write_buff and read_buff
3.  Create a socket for TCP communication using socket(AF_INET, SOCK_STREAM, 0)
    If failed → print error and exit
4.  Set up the server address structure with appropriate values
    for family, port and IP address
5.  Connect to the server using connect()
    If failed → print error and exit
6.  Initialize flag = 1
7.  For i = 0 to total frames:
    7.1  Clear buffers using memset()
    7.2  If i == 5 and flag == 1:
         → Simulate packet loss
         → Receive frame but discard it
         → Print "Packet lost"
         → Set flag = 0
         → Decrement i and continue
    7.3  Receive frame from server using recv()
    7.4  Check if frame is in correct order
         If out of order:
            → Print "Discarded out-of-order frame"
            → Decrement i
         If correct order:
            → Print received frame
            → Prepare ACK message
            → Send ACK to server using send()
            → Print ACK sent
8.  Close socket using close()
9.  Stop


KEY RULE:
---------
If frame N is lost →
Retransmit N, N+1, N+2 ... ALL frames from N onwards


ROLES:
------
Server = SENDER
Client = RECEIVER
